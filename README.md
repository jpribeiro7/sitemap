# Sitemap Generator

This REST-Api uses the Nest framework which forces an MVC pattern.
It runs on port 3000

## Install
npm install
## How to start
npm run start:dev
## How to test
npm run test

## Routes 
There is only one route, /sitemap, which accepts a query param url
## Request Example
GET localhost:3000/sitemap?url=https://pcdiga.com
